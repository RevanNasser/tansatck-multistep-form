# TanStack Form Multi-Step Demo

A **demo project** showcasing a **multi-step form** implementation using **TanStack Form**, **Zod** for schema validation, and **ShadCN UI components**. This project demonstrates a clean and interactive **phone number + OTP verification flow** with **dark/light mode toggle** and responsive UI.

---



## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tanstack-form-demo.git
cd tanstack-form-demo
```
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open http://localhost:3000 to view the project
